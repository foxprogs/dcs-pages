How do I start?

First register for a Door43 account. This account can both be used here on DCS and in apps that connect to DCS

Then we suggest you install Scribe or translationCore to start translating. Both applications allow you to synchronize your work with DCS.

About DCS

Door43 Content Service (DCS) is an online repository for open-licensed Biblical content in any language.

DCS is an open project that is managed by unfoldingWord and many volunteers.

Every change made in DCS is automatically being processed into a web page on Door43.

What tools are available?

If you are a translator, find out more about our Bible translation tools.

To learn how to translate, check, publish, and use our tools, check out translationAcademy or view the DCS knowledgebase articles.

If you are a developer, you can connect your own tool or script to the DCS API. We have already prepared a Python DCS API client for you.
